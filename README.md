# wifihack (uso educativo)

Este proyecto es un **script educativo** para aprender:
- Bash scripting
- Uso de la terminal en Termux
- Automatización básica de tareas

⚠️ **Advertencia**
Este script es solo para fines educativos.
NO debe usarse en redes, sistemas o dispositivos sin autorización.
El autor no se hace responsable del uso indebido.

## Requisitos
- Termux o Linux
- Bash
- Git

## Instalación

Clona el repositorio:

git clone https://github.com/HACKEX7/wifihack.git

Entra a la carpeta:

cd wifihack

Da permisos de ejecución:

chmod +x wifihack.sh

## Uso

Ejecuta el script con:

./wifihack.sh

O alternativamente:

bash wifihack.sh

## Notas
- Revisa el código antes de ejecutarlo
- Modifícalo para aprender
- Úsalo solo en entornos de prueba o con permiso
